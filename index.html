⚡ experto — algoritmo de búsqueda manual nivel 4
Búsqueda binaria en lecturas ordenadas
Implementá una búsqueda binaria (sin usar funciones de Python como in o index()) sobre esta lista ya ordenada:
niveles = [33.1, 37.4, 41.2, 44.8, 46.3, 48.0, 49.7, 51.2, 53.8, 55.1, 57.3, 59.9]

El usuario ingresa un nivel a buscar. El algoritmo:
  • Define izq = 0, der = len(niveles) - 1
  • En cada ciclo calcula medio = (izq + der) // 2 y compara
  • Si el valor del medio es el buscado → encontrado en posición X
  • Si el buscado es mayor → izq = medio + 1
  • Si es menor → der = medio - 1
  • Muestra en cuántos pasos lo encontró (o que no existe)
Luego mostrá cuántos pasos necesitaría una búsqueda lineal (for normal) para comparar.
La búsqueda binaria funciona dividiendo el espacio a la mitad cada vez. En 12 elementos, a lo sumo necesita 4 pasos. Una búsqueda lineal necesitaría hasta 12.

Ordenar lecturas con Bubble Sort
Implementá el algoritmo Bubble Sort sin usar sorted() ni .sort() para ordenar lecturas de caudal:
caudales = [7.3, 2.1, 9.8, 4.5, 6.2, 1.7, 8.4, 3.9, 5.6, 0.8]

El algoritmo:
  • Usa dos for anidados para comparar pares adyacentes
  • Si el elemento actual es mayor al siguiente, los intercambia
  • Muestra la lista después de cada pasada completa
  • Cuenta cuántos intercambios hizo en total
  • Al final compara el resultado con sorted() para verificar que sea correcto
Después repetí el proceso pero ordenando de mayor a menor (solo cambiá la condición de comparación).
Para intercambiar dos elementos: lista[i], lista[i+1] = lista[i+1], lista[i]

Sistema completo de gestión de la Represa El Calafate
Este es el ejercicio más completo. Creá un programa de gestión de la represa con menú de opciones que permanece activo hasta que el usuario elija salir. El menú tiene estas opciones:

1. Registrar turno — Pide nivel (validado 0–60 m), caudal entrante y caudal saliente. Los guarda en 3 listas paralelas. Calcula la variación neta del nivel.
2. Ver resumen del día — Muestra todos los turnos registrados con un número de fila. Calcula nivel mínimo, máximo y promedio del día. Muestra cuántos turnos tuvieron nivel crítico (>50 m).
